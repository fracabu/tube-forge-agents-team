### PAGINA 2: Le 5 Automazioni con Checklist

#### AUTOMAZIONE 1: Email Intelligenti
**Tempo risparmiato: 2 ore/settimana | Costo: 0-20 EUR/mese**

```
[ ] Creare account Zapier gratuito (zapier.com)
[ ] Collegare il tuo account Gmail a Zapier
[ ] Creare account ChatGPT (chat.openai.com)
[ ] Cercare template "Gmail to ChatGPT" su Zapier
[ ] Configurare le parole chiave trigger (es: "preventivo", "orari", "disponibilita")
[ ] Scrivere il prompt di sistema con info azienda
[ ] Testare con email di prova
[ ] Attivare l'automazione

LINK DIRETTO: zapier.com/apps/gmail/integrations/chatgpt
```

---

#### AUTOMAZIONE 2: Fatturazione Automatica
**Tempo risparmiato: 1-2 ore/settimana | Costo: 6-15 EUR/mese**

```
[ ] Registrarsi su Fatture in Cloud (fattureincloud.it)
[ ] Attivare prova gratuita 15 giorni
[ ] Importare anagrafica clienti esistente
[ ] Creare prima fattura modello per cliente ricorrente
[ ] Impostare frequenza (mensile/trimestrale)
[ ] Configurare invio automatico o salva come bozza
[ ] Attivare solleciti automatici pagamenti
[ ] Collegare conto bancario per riconciliazione

LINK DIRETTO: fattureincloud.it/registrazione
```

---

#### AUTOMAZIONE 3: Social Media AI
**Tempo risparmiato: 2-3 ore/settimana | Costo: 0-29 EUR/mese**

```
[ ] Creare account Canva (canva.com)
[ ] Valutare upgrade a Canva Pro (110 EUR/anno)
[ ] Accedere a ChatGPT
[ ] Generare piano editoriale mensile con prompt:
    "Crea 20 idee di post per [SETTORE] pensate per [CLIENTE IDEALE]"
[ ] Salvare idee in documento
[ ] Creare primo post con Magic Design di Canva
[ ] Usare Magic Write per copy
[ ] Configurare Content Planner
[ ] Schedulare prima settimana di contenuti

PROMPT CHATGPT BONUS:
"Sei un social media manager esperto. Crea un piano editoriale
di 4 settimane per [TIPO ATTIVITA]. Per ogni post indica:
tipo di contenuto, copy suggerito, hashtag consigliati."
```

#### AUTOMAZIONE 4: Trascrizione Riunioni
**Tempo risparmiato: 1-2 ore/settimana | Costo: 0-10 EUR/mese**

```
[ ] Creare account Otter.ai (otter.ai)
[ ] Collegare calendario Google/Outlook
[ ] Installare estensione browser Otter
[ ] Configurare Otter per Zoom/Meet/Teams
[ ] Fare riunione di prova (anche con te stesso)
[ ] Verificare qualita trascrizione
[ ] Esplorare funzione "Key Takeaways"
[ ] Condividere prime note con un collega

ALTERNATIVA GRATUITA:
[ ] Usare Whisper (openai.com/whisper)
[ ] Registrare audio riunione
[ ] Caricare su servizio Whisper online
[ ] Scaricare trascrizione

LINK DIRETTO: otter.ai/signup
```

```

[ ] Creare account ManyChat (manychat.com)

Scrivi LISTINO per ricevere i nostri prezzi.
```
